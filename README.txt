This project is about a lead programmer for the Italian restaurant Basta Fazoolin' with My Heart and his or her main responsability is to make sure that everything is organized. 


The first class in the project, the Menu class, contains standard methods and a custom method to calculate the bill for some items on a menu. Some dictionaries with items are initialized in order to be able to call the menu class and define the menus available.


The 2nd class, which is the Business class, is constructed and it will be used to initiate a business with 2 franchises.


The 3rd class, which is the Franchise class, has standard methods and a custom method to show the available menus at a certain time of the day.
The 3rd class is called to initialize franchises.



At the end of the project, a new menu with new items is created, along with a franchise and a business.
